# Void MUD Public Release Notes

This repository publishes the public, spoiler-safe changelog for Void MUD. It highlights player-visible updates such as new systems, UI/UX improvements, and stability fixes while keeping secrets undisclosed.

## What to expect
- Summaries focus on features and quality-of-life updates that players can discover naturally.
- Hidden areas, puzzles, and rewards remain unnamed; late-game surprises stay secret.
- See `CHANGELOG.md` for the latest entries.
